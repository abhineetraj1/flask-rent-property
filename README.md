# Flask Rent property

I have created a property rental platform enabling users to register and manage their properties for lease. The backend development is implemented using Flask, while MongoDB serves as the database management system. Within the 'rentproperty' database, two collections are established: 'users' and 'properties'. The 'users' collection contains fields for id, username, email, password, phone number, and name. Meanwhile, the 'properties' collection includes id, username, property name, owner's phone number, rent price, address, city, area, number of bathrooms and bedrooms, image URL, and additional details. Prospective tenants can utilize a user-friendly HTML/CSS frontend to search for available properties based on location.


## Installation
1) First install python3.6 in your respective system
2) Download this source code
3) Download and setup Mongosh and MongoDB Compass
4) Open MongoDB Compass and setup the database `rentproperty` with two collections - `properties` and `users`
5) Setup the attributes as given in description

### For macOS

Open terminal , execute the below code

```
pip3 install flask
```

### For linux

Open terminal , execute the below code

```
sudo pip3 install flask
```

### For windows

Open command prompt (search for this, you'll get it) , execute the below code

```
pip3 install flask
```


## Execution

### For macOS

1) Open terminal in the respective folder (where you have downloaded the project) write the below code and hit enter

```flask run```

2) And then open web browser and go to -> http://127.0.0.1:5000

### For Linux

1) Open terminal in the respective folder (where you have downloaded the project) write the below code and hit enter

```flask run```

2) And then open web browser and go to -> http://127.0.0.1:5000


### For Windows

1) Download this source code and double click on file -> app.py .

2) And then open web browser and go to -> http://127.0.0.1:5000
